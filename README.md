# Assignment 1: Option 9

In this project, we designed a compare function in three languages (Java, Python, C++), and then visualized the partial-ordering as Hasse diagrams. The code included in this repo formats everything as a .DOT file/terminal output that can be copied into GraphViz for visualization.
